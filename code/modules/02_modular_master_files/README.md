# 02_modular_master_files

This stage joins each RCRAInfo module into one analysis-ready master file. It
holds a single `rcrainfo/` subfolder because every master so far is built from
RCRAInfo; a different data system would get its own sibling folder here.

See [rcrainfo/README.md](rcrainfo/README.md) for the scripts, the join design,
and the outputs.
