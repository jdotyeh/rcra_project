# 03_panels

Note: This folder has been verified (Jul 19).

This stage builds the facility panels, the analytical endpoint of the project.
It holds a single `rcrainfo/` subfolder because every panel so far is built from
the RCRAInfo master files and the Biennial Report; a different data system would
get its own sibling folder here.

See [rcrainfo/README.md](rcrainfo/README.md) for the panels, the shared function
file, the build shortcut, and the institutional context behind the panel design.
