# eGRID Data

EPA Emissions & Generation Resource Integrated Database (eGRID) workbooks,
downloaded by `code/modules/01_download/egrid/01_download_data.R`. One workbook
per year in this folder, kept raw as downloaded.

| File | What it includes |
|------|------------------|
| `EGRID_PLANT_<year>.xlsx` | Annual eGRID data workbook. Plant-level emissions and generation, plus sheets for units, generators, states, balancing authorities, and a US summary. |
